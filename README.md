# NxtPeople Interactive 3D Ecosystem Twin

A highly polished, interactive 3D visualization of the NxtPeople HR ecosystem built with **Three.js**. This digital twin serves as a premium presentation and onboarding tool, visually mapping the flow of data and operations across all NxtPeople modules.

## 🚀 Features

*   **Interactive 3D Engine:** Navigate the entire NxtPeople ecosystem in a smooth, hardware-accelerated 3D environment.
*   **Cinematic Auto-Play Demo:** A fully automated demonstration mode that pans the camera through each step of the recruitment lifecycle, highlighting data flow and module capabilities.
*   **Module-Specific Branding:** Exact, pixel-perfect brand color synchronization for each node (`#1D5CE5` Hub, `#CB3838` Job, `#E03C8A` Campaign, etc.).
*   **Dynamic Information Flow:** Accelerated particle animations along connection paths visually demonstrate data transferring between active modules.
*   **Premium Glassmorphism UI:** Modern, translucent overlays provide detailed module descriptions without obscuring the 3D scene.
*   **Custom Rendering:** Advanced post-processing with `UnrealBloomPass` for neon core glow effects and native canvas drawing for the official NxtPeople logo.

## 🧩 Modules Included

The digital twin maps the entire HR lifecycle from candidate attraction to payroll:

1.  **NxtPeople (Hub)** - Centralized backbone & unified API.
2.  **NxtJob** - AI-powered vacancy creation & multiposting.
3.  **NxtCampaign** - Multi-channel automated recruitment marketing.
4.  **NxtForm** - Mobile-first data collection and e-signatures.
5.  **NxtMatch** - AI screening and smart shortlisting.
6.  **NxtPlan** - Workforce scheduling and shift management.
7.  **NxtPay** - Automated payroll processing and invoicing.
8.  **NxtInsight** - Real-time BI & analytics dashboards.

## 🛠️ Technology Stack

*   **HTML5 / CSS3:** Structure and modern, responsive styling.
*   **Three.js:** The core 3D WebGL rendering engine.
*   **3D-Force-Graph:** Force-directed graph physics integration.
*   **Vanilla JavaScript:** Lightweight, dependency-free logic and animation loop.

## 💻 Running Locally

Because this is a pure frontend application, simply open `index.html` in any modern web browser to view the visualization. No build step or package manager is required!
